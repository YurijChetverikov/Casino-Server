Обязательные файлы:
pl.bin - файл с информацией о количестве игроков, их имён, паролей и прочего + информация об добавленных играх

Файлы данных:
pl0.dat - файл с данными об игроке: транзакциями, сыгранными играми (вместо 0 в названии файла - ID игрока)
big_data.dat - файл со всеми транзакциями
pic.dat - файл с фотографией казина, там просто битмап
